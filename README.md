<h1>TalentDash</h1>
TalentDash is a salary intelligence platform built with Next.js 15, TypeScript, and Tailwind CSS. It helps software engineers explore, compare, and analyze compensation data across leading technology companies.

<h1>Overview</h1>
The platform provides a fast and searchable salary database where users can:
<ul>
  <li>Explore software engineering salaries across multiple companies.</li>
  <li>Filter compensation data by company, role, level, and location.</li>
  <li>Compare salary offers side-by-side.</li>
  <li>View company-specific compensation insights.</li>
  <li>Analyze salary distributions across engineering levels.</li>
  <li>Access SEO-optimized salary pages for discoverability</li>
</ul>
